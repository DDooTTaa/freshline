# 언어 스트레칭 (Language Stretching)

시인의 시 쓰는 방식을 참고한 창작 앱입니다.

## 기능

1. **랜덤 단어 제공**: 랜덤으로 3개의 단어가 제공됩니다
2. **문장 작성**: 제공된 단어들의 특징을 살려 문장을 작성합니다
3. **단어 교체**: 원하는 단어를 다른 단어로 바꿔볼 수 있습니다
4. **작품 저장**: 작성한 작품을 로컬에 저장할 수 있습니다
5. **작품 모아보기**: 저장된 작품들을 갤러리에서 확인할 수 있습니다
6. **공유 기능**: 작성한 작품을 클립보드에 복사하여 공유할 수 있습니다

## 시작하기

### 필요 조건

- Flutter SDK (3.0.0 이상)
- Dart SDK

### 설치 및 실행

```bash
# 의존성 설치
flutter pub get

# 앱 실행
flutter run
```

## 프로젝트 구조

```
lib/
├── main.dart                 # 앱 진입점
├── models/                   # 데이터 모델

│   └── creation.dart        # 작품 모델

    ├── word_service.dart    # 단어 관련 서비스

```

## 주요 기능 설명

### 랜덤 단어 제공
- 80개 이상의 한국어 단어 풀에서 랜덤으로 3개를 선택합니다
- 새로고침 버튼으로 새로운 단어를 받을 수 있습니다

### 문장 작성
- 제공된 단어들의 특징을 살려 자유롭게 문장을 작성합니다
- 실시간으로 문장을 수정할 수 있습니다

### 단어 교체
- 각 단어를 탭하여 유사어로 교체할 수 있습니다
- 단어별로 제공되는 유사어 목록에서 선택합니다

### 저장 및 관리
- SQLite를 사용하여 로컬에 작품을 저장합니다
- 저장된 작품은 갤러리에서 확인하고 수정/삭제할 수 있습니다

## 기술 스택

- **Flutter**: 크로스 플랫폼 앱 개발 프레임워크
- **SQLite**: 로컬 데이터베이스 (sqflite 패키지 사용)
- **Shared Preferences**: 간단한 설정 저장 (향후 확장용)
